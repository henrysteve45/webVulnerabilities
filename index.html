Information Your Browser Has About You

This web application demonstrates a few pieces of information that your browser knows about you. All of this information can be potentially accessed by malicious sites without your permission.

How is this possible?

Developers can use the navigator.userAgent JavaScript property to list information about the user's browser, OS, etc.

Duck typing is a technique used to identify user information based on the duck test (if it walks, looks, and sounds like a duck... then it IS a duck). Threat actors can spoof their user agent, duck typing combats this by identify browsers based on known quirks they have.

The navigator.getBattery() property can be used to view a device's battery percentage. This is an deprecated feature removed from most browsers due to security concerns.

Mitigation techniques

User agent information can be hidden by browser extensions such as NoScript.

How is this possible?

Cloudflare can be used to get a user's public IP address. Your public IP address is provided by your router/internet service provider and connects everyone in your local area network (household or office) to the internet. It may look something like this: 92.123.50.225. A private IP address is used for communications between devices within your own network. It is not recognized on the internet and looks something like 192.168.1.240.

There are browser methods such as navigator.connection that gives websites access to this information.

Mitigation techniques

Use a VPN service. VPNs allow users to use unsecured public networks like a secure private network by encrypting all data between the remote and client device.

Use the Tor browser. This browser allows internet users to browse anonymously online and keep their activities hidden from websites, ISPs, and advertisers.
